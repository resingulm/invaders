# Quickstart: Testing the Game Lifecycle UI

This guide explains how to test the new splash screen and game-over UI.

## Testing the Splash Screen

1.  Run the game.
2.  Observe that a splash screen is displayed for 5 seconds before the main game starts.

## Testing the Game Over Screen

1.  Play the game until you lose all your lives or the aliens reach the bottom of the screen.
2.  Observe that a game-over screen is displayed with "Restart" and "Quit" buttons.
3.  Click the "Restart" button and observe that the game restarts.
4.  Play the game again and reach the game-over screen.
5.  Click the "Quit" button and observe that the browser tab closes.
